Passo 1:criando o arquivo Lottie

caso precise criar um arquivo angular novo:

$ ng new testando-lottie --skip-tests

Acessando o

Lottie

na seção de animações gratuitas, em seguida selecione uma animação de sua escolha, baixe na opção 'lottie JSON' dando o nome que achar melhor, exemplo: animation.json.

Crie uma pasta chamada lottie dentro da sua pasta 'src/assets', dentro dessa pasta onde você irá localizar e guardar suas animações

Passo2:Criando sua estrutura para exibir sua animação

Para isso precisamos criar um componente que vai redenrizar nossa animação, com o comando:

$ ng g c shared/nome-de-componente-da-sua-escolha

Chamando seu componente na

app.component.html

Agora sua pasta do componente que foi criado, vá até

nome-de-componente-da-sua-escolha.component.scss

e adicione esse estilo

:host

height: 100%;

overflow: hidden;

display: flex;

flex-flow: column nowrap;

align-items: center;

justify-content: center;

position: relative;

}

Passo3:Instalando pacotes Lottie

Usando o comando do npm installa o pacote Lottie

$ npm install lottie-web ngx-lottie

Agora vamos adicionar o lottie ao arquivo

app.module.ts

importando Lottie e o CommonModule

Passo4:Adicionar a animação Lottie ao componente de tela Inicial

No arquivo

nome-de-componente-da-sua-escolha.component.TS

, criando uma variavel que recebe o caminho da animação direto do assets e uma nova função que se vincula ao evento

Iremos agora para o arquivo HTML do componente criado e criaremos o evento dentro de uma div
